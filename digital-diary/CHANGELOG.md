# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

> **The Digital Diary is now feature-complete for everyday personal journaling. Future development will be guided by real usage, not planned feature expansion.**

## [1.2.1] - 2026-07-18

### Added
- Memory context menu
- Undo delete
- Inline rename
- Keyboard shortcuts

### Changed
- Optimistic UI updates
- Autosave status
- Search improvements

### Fixed
- SSR hydration issues
- Client/server component boundaries

## [1.2.0] - 2026-07-16

### Added
- Complete memory lifecycle (Create, Edit, Delete, Duplicate, Change Date)
- Safe GitHub transaction layer
- Resilient date fallback model

## [1.1.0] - 2026-07-10

### Added
- Authentication & security
- GitHub OAuth Integration
- Middleware route protection

## [1.0.0] - 2026-07-01

### Added
- Foundation (GitHub storage, editor, synchronization)
- Visual Shell and Theme
- Calendar Timeline and Organization
